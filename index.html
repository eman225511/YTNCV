How to use:
Paste any YouTube video, Shorts, or playlist link.

Works with:
• youtube.com
• youtu.be
• playlists (?list=)

You can also open directly with:
?yt=YOUR_LINK

See the full GitHub README for more info.

YTNCV Bookmarklet
  • YTNCV Watcher (Multi): Get Shortcut — Prompts Safari, Chrome, or Web View
  • YTNCV Watcher (BasicS): Get Shortcut — Opens directly in Safari
  • YTNCV Watcher (BasicC): Get Shortcut — Opens directly in Chrome
